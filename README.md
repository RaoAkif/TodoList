# Todo List
> A very simple todo app with CRUD operations.

## Built With
- HTML, CSS, JS, Node, Webpack

## Live Version
- [Live Link](https://raoakif-todojs.netlify.app/)

### Prerequisites
Node JS

### Setup
`git clone git@github.com:RaoAkif/TodoList.git`

### Install
- `cd TodoList`
- `npm i`

### Usage
`npm start`

### Run tests
`npm test`
